# 商品与股票量化交易系统

## 项目目的
本项目旨在探索、评估并最终搭建一个面向国内金融市场的量化交易系统。
系统初期的核心聚焦于**商品（期货/期权）的量化交易**，主要基于分钟级别（K线）的交易逻辑，不涉及Tick级别的高频交易。
在未来，系统计划横向扩展，延伸成为一个兼容**股票市场的综合性量化交易平台**。

## 建设路径
1. **前期调研**：深入研究市面上主流的开源与商业量化框架（如VNPY、天勤TqSdk、通达信系列、同花顺系列等），从源代码架构、生态、回测及实盘能力等多个维度进行优劣分析。
2. **技术选型**：基于分钟级策略的需求，结合后续拓展股票市场的目标，决定是采用现成框架（如VNPY、QMT等）还是借鉴优秀开源项目的架构进行完全自主研发。
3. **系统开发与回测**：完成基础数据源接入、策略回测引擎的搭建或适配。
4. **实盘对接**：通过标准的交易接口（如CTP用于商品，QMT/通达信用于股票）实现自动化交易。
